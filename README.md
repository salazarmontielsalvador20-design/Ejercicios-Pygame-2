Graficación - Ejercicios con Pygame

Este repositorio contiene los ejercicios y mini-proyectos desarrollados para la materia de Graficación. El objetivo es aplicar los conceptos de animación, manejo de sprites, colisiones y creación de juegos simples utilizando la biblioteca Pygame en Python.

Contenido del Repositorio

El proyecto está organizado en carpetas, una por cada sesión de la asignación "Ejercicios Pygame 2".

/Sesion4

sesion4_ej1.py: Ejercicio de rebote básico con velocidad variable.

sesion4_ej2.py: Animación de un círculo con efecto de pulsación (crece y decrece).

sesion4_ej3.py: Simulación de gravedad y rebote con pérdida de energía.

/Sesion5

sesion5_ej1.py: Carga de una imagen (nave.png) con ajuste de tamaño dinámico usando las teclas + y -.

sesion5_ej2.py: Animación de sprites cargando 4 imágenes diferentes (nave.png, bola.png, robot.png, meteorito.png) como fotogramas.

sesion5_mini.py: Mini-proyecto. Una nave (nave.png) que rota para apuntar al cursor del ratón y avanza.

/Sesion6

sesion6_ej1.py: (Ejercicio básico) Colisión simple que cambia el color de un objeto.

sesion6_ej2.py: (Recolector de Círculos) Juego de un rectángulo que recoge círculos que reaparecen aleatoriamente.

sesion6_ej3.py: (Esquiva los Círculos) Juego de un rectángulo que evita círculos móviles. El juego termina al colisionar.

sesion6_mini.py: Mini-proyecto. Juego simple de una nave que recoge "puntos" (bola.png) y evita "obstáculos" (obstaculo.png).

/ (Raíz del proyecto)

nave.png: Asset de imagen para el jugador.

bola.png: Asset de imagen para puntos/items.

obstaculo.png: Asset de imagen para obstáculos.

robot.png: Asset de imagen (usado en sesion5_ej2).

meteorito.png: Asset de imagen (usado en sesion5_ej2).
